Bienvenido al Editor de FAQs

Categoría: Es una carpeta para agrupar FAQs relacionadas. Una categoría puede contener otras categorías o FAQs.

FAQ: Es una pregunta y su respuesta. Las FAQs viven dentro de las categorías.

Para empezar: Carga un archivo `faqs.json` usando el botón de abajo a la izquierda para comenzar a editar.

El ID es obligatorio para las FAQs.
